Mobile Application Development
WebView in Android Apps makes development possible to create a Native Application that can communicate with mobile hardware using just only HTML, CSS and Javascript so we can defined it as Hybrid Android App. People like me in web development field can jump right in just using HTML5, CSS, Javascript to create an Android Apps and having to get familiar about java language little bit at a time. IDE used for the practices was Netbeans 7.1.2 with NBAndroid Plugins, JQuery Mobile or Bootstrap for the apps. PhoneGap for the hardware connector used for the development process. API, Json and RESTFul web service used for data retrieval or just plain HTML5 data storage. Mobile Application Development is possible with all the mobile handset that have webview / UiWebView property. And now React Native showed me another level of mobile aplication development.

Some companies already started to focusing on the products to make the future of Connected Things. For me this a step up for challenge to get more advantage with the used of internet on different medias. My fun projects with the used of IOT :
